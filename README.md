# 🚀 IT Support to SRE / DevOps & Backend Roadmap Tracker

A minimalist, interactive **To-Do List Tracker** designed specifically for tracking career-switch progress from IT Support to SRE/DevOps and Backend Developer.

🌐 **Live Demo:** [https://roadmap-todo-pearl.vercel.app](https://roadmap-todo-pearl.vercel.app)

## 🎯 The Roadmap Overview (18–24 Months)

This roadmap is aligned with the official tracks from [roadmap.sh](https://roadmap.sh/) and tailored for a structured career switch.

1. **Phase 1: Foundation (Bulan 1–3)**
   - Linux, Docker, Python, Git.
   - *Target:* FastAPI + PostgreSQL running in Docker, basic CI/CD pipeline.
2. **Phase 2: Cloud & Node.js (Bulan 4–6)**
   - Node.js, K8s Basics, AWS.
   - *Target:* Pass AWS SAA, local Kubernetes cluster running.
3. **Phase 3: The SRE Way (Bulan 7–10)**
   - Go, Terraform, Ansible, Prometheus/Grafana.
   - *Target:* Build high-perf Go API + AWS Infrastructure via Terraform.
4. **Phase 4: Enterprise Ready (Bulan 11–14)**
   - Java, Spring Boot, GitOps (ArgoCD).
   - *Target:* Helm deployments and Staging/Prod workflows with ArgoCD.
5. **Phase 5: Microservices (Bulan 15–20)**
   - Integrating Python, Node, Go, and Java APIs into one Polyglot System.
   - *Target:* Full Kubernetes deployment & advanced system testing.
6. **Phase 6: Final Polish (Bulan 20–24)**
   - Data Structures, Algorithms, System Design, and Mock Interviews.
   - *Target:* Land the Offer 🎯.

## ✨ Features
- **Interactive Checklists:** Track your progress per phase.
- **Local Storage Saving:** Your progress won't be lost when you refresh or close the tab.
- **Progress Bar:** Real-time percentage completion calculation.
- **Direct Resources:** Links directly to the exact curriculum on [roadmap.sh](https://roadmap.sh/).

## 🛠️ Tech Stack
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Deployment:** Vercel

## 💻 Getting Started (Local Development)

Clone the repository and install the dependencies:

```bash
git clone https://github.com/arighmt67-bit/roadmap-todo.git
cd roadmap-todo
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
